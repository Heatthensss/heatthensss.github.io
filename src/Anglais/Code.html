Coding Project :

JO Paris

JO Paris is a school project created in laravel. You can create an account and log in. Once logged in, you can create a sport, but only the creator can modify it, delete it and add an image. There are also administrator accounts that allow you to modify and delete all sports.

Desert crossing

Desert Crossing is a school project in Python with no graphical interface. The goal is to cross the desert without getting caught. To do this, you have several options, as you have to watch out for the camel's thirst and fatigue. The game ends when the player has covered 300km or has been caught.

Flatcraft

Flatcraft is a java fx school project. The aim is to recreate a 2d minecraft, with a given base. It has the same dimensions as minecraft.

Web Marathon

The Web Marathon is an event organized by the IUT in Lens, where MMI and Computer Science students work in groups for 2 days. The aim was to create a site where you can create and read stories in which you are the hero.
